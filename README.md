How do I set query?
-------------------
```java
     VolleyHelper volleyHelper=new VolleyHelper(this);
     volleyHelper.setQuery("key","value");
```

How do I set token?
-------------------
```java
   UtilityHelper utilityHelper=new UtilityHelper(this);
   utilityHelper.setSharedPreferences("token"," Your token ");
```
